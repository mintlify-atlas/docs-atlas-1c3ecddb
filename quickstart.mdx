## Fast path to your first order extraction

This guide will have Chat2Cash extracting orders from WhatsApp chats in under 5 minutes using Docker Compose.

<Steps>
  <Step title="Prerequisites">
    Before you begin, ensure you have:
    
    - **Docker Desktop** or Docker Engine with Docker Compose installed
    - An **Anthropic API key** ([Get one here](https://console.anthropic.com/))
    - Git installed on your machine
    
    <Info>
      Docker Compose will automatically set up PostgreSQL, Redis, the API server, and 2 worker replicas.
    </Info>
  </Step>

  <Step title="Clone and configure">
    Clone the repository and set up your environment:
    
    ```bash
    git clone https://github.com/yourusername/chat2cash.git
    cd chat2cash
    ```
    
    Create a `.env` file in the root directory:
    
    ```bash .env
    ANTHROPIC_API_KEY="sk-ant-api03-..."
    DEFAULT_BUSINESS_NAME="My Saree Shop"
    DEFAULT_GST_NUMBER="22AAAAA0000A1Z5"
    SENTRY_DSN="https://your-sentry-dsn@sentry.io/..."  # Optional
    ```
    
    <Warning>
      Replace `sk-ant-api03-...` with your actual Anthropic API key. The system will not start without it.
    </Warning>
  </Step>

  <Step title="Start the stack">
    Launch all services with a single command:
    
    ```bash
    docker-compose up --build
    ```
    
    Wait for the health checks to pass. You should see:
    
    ```bash
    ✓ db (healthy)
    ✓ redis (healthy)
    ✓ api (started)
    ✓ worker (started, 2 replicas)
    ✓ frontend (started)
    ```
    
    The API will be available at `http://localhost:3000`
    
    The frontend will be available at `http://localhost:5173`
  </Step>

  <Step title="Test your first extraction">
    Extract an order from a sample chat message:
    
    ```bash
    curl -X POST http://localhost:3000/api/extract \
      -H "Content-Type: application/json" \
      -d '{
        "sender": "+919876543210",
        "message": "bhaiya 2 kilo aaloo aur 1 kg tamatar bhej do kal tak"
      }'
    ```
    
    You should receive a structured JSON response:
    
    ```json
    {
      "items": [
        {
          "name": "Potato (Aaloo)",
          "quantity": 2,
          "unit": "kg"
        },
        {
          "name": "Tomato (Tamatar)",
          "quantity": 1,
          "unit": "kg"
        }
      ],
      "delivery_date": "2026-03-05",
      "intent": "order"
    }
    ```
    
    <Check>
      If you see structured order data, Chat2Cash is working correctly!
    </Check>
  </Step>
</Steps>

## What just happened?

Docker Compose launched a distributed system with:

- **PostgreSQL** database for order storage
- **Redis** with AOF persistence for job queues
- **API server** handling HTTP requests
- **2 Worker replicas** processing AI extraction jobs in parallel
- **Frontend** React application with Vite

The `/api/extract` endpoint queued your message, a worker picked it up, Claude analyzed the Hinglish text, and returned structured order data.
