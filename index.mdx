    <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
    <p className="text-lg text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get Chat2Cash running in minutes with Docker or deploy manually for custom setups
    </p>
  </div>

  <Steps>
    <Step title="Install with Docker Compose">
      Clone the repository and start the full distributed stack with PostgreSQL, Redis, API server, and background workers:

      ```bash
      git clone https://github.com/swayam-mishra/Chat2Cash.git
      cd Chat2Cash
      ```

      Create a `.env` file with your credentials:

      ```bash
      ANTHROPIC_API_KEY="sk-ant-api03-..."
      DEFAULT_BUSINESS_NAME="My Business"
      DEFAULT_GST_NUMBER="22AAAAA0000A1Z5"
      ```

      Start the stack:

      ```bash
      docker-compose up --build
      ```

      The API will be available at `http://localhost:3000`
    </Step>

    <Step title="Extract your first order">
      Send a message to extract order details from a Hinglish conversation:

      ```bash
      curl -X POST http://localhost:3000/api/extract \
        -H "Content-Type: application/json" \
        -d '{
          "message": "Bhaiya 2 kilo aaloo aur 3 kilo pyaaz chahiye, kal delivery karna"
        }'
      ```

      <Accordion title="Example response">
        ```json
        {
          "id": "ord_abc123",
          "items": [
            {
              "name": "Potato",
              "quantity": 2,
              "unit": "kg"
            },
            {
              "name": "Onion",
              "quantity": 3,
              "unit": "kg"
            }
          ],
          "confidence": 0.95,
          "status": "pending",
          "deliveryDate": "2024-03-05",
          "createdAt": "2024-03-04T10:30:00Z"
        }
        ```
      </Accordion>
    </Step>

    <Step title="Generate GST invoice">
      Convert the extracted order into a professional, GST-compliant PDF invoice:

      ```bash
      curl -X POST http://localhost:3000/api/generate-invoice \
        -H "Content-Type: application/json" \
        -d '{
          "orderId": "ord_abc123"
        }'
      ```

      The response includes the invoice with CGST/SGST/IGST breakdowns and sequential numbering.
    </Step>

    <Step title="Explore advanced features">
      Check out async processing, multi-tenant setup, and real-time job tracking with SSE streaming. See the [Architecture](/architecture/overview) section for distributed system details.
    </Step>
  </Steps>
</div>
